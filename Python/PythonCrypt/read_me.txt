PythonCrypt – программа, которая шифрует опредленные файлы в заданной пользователем директории с помощью метода шифрования AES
Программа имеет две версии: 1. Консольная(Сonsole_version_of_Python_Crypt_5.py 2. Оконная(PythonCrypt_Window_Application.py)
-----
PythonCrypt is a program that encrypts specific files in a user-defined directory using the AES encryption method.
The program has two versions: 1. Console (Console_version_of_Python_Crypt_5.py 2. Window (PythonCrypt_Window_Application.py)
